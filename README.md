EXPENSE TRACKER – SMART FINTECH APP


The Expense Tracker Smart Finance App is a full-stack personal finance management system designed to help users track income, expenses and financial habits in real time. It goes beyond a normal tracker by adding AI-powered insights, financial personality classification like charecter based as a unique feature 

The system analyzes user spending patterns and converts financial data into simple insights, visual charts and a gamified financial character system.
