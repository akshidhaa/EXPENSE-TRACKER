💸 EXPENSE TRACKER – SMART FINTECH APP
🧠 Overview

The Expense Tracker Smart Finance App is a full-stack personal finance management system designed to help users track income, expenses, and financial habits in real time. It goes beyond a normal tracker by adding AI-powered insights, financial personality classification, and interactive analytics dashboards.

The system analyzes user spending patterns and converts financial data into simple insights, visual charts, and a gamified financial character system.

🎭 Financial Character System (Unique Feature)

The app assigns a financial personality based on spending behavior:

🐢 Saver Turtle → Safe & low spender
🦅 Smart Eagle → Balanced spending
🔥 Risk Tiger → High spending warning
🧠 Finance Master → Excellent financial control

👉 Updates dynamically based on income-to-expense ratio

Provides:
